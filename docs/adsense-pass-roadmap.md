# Tourpick360 AdSense Pass Roadmap

## Positioning

승인 전 Tourpick360은 잡블로그가 아니라 하나의 좁은 주제를 깊게 다루는 정보 사이트로 운영한다.

Core niche:

```txt
국내 계절 축제 여행 가이드
```

Expanded explanation:

```txt
국내 여행지 중에서도 계절별 축제, 숙소 위치, 이동 동선, 주차, 이용 시간, 가족/커플/부모님 여행 기준을 정리하는 전문 정보 사이트
```

## Approval Formula

### 1. Category Compression

승인 전에는 카테고리를 넓히지 않는다.

Visible categories:

```txt
국내여행
제주 집중 가이드
여행 준비
```

운영 방식:

- 실제 핵심 글은 국내여행 아래에 몰아준다.
- 제주 글 10개를 첫 전문 묶음으로 완성한다.
- 이후 부산, 강릉, 여수, 경주, 서울은 국내여행 안에서 확장한다.
- 승인 전에는 해외여행, 맛집, 쇼핑, IT 등 주제 이탈 카테고리를 만들지 않는다.

### 2. Article Formula

모든 승인용 글은 아래 구조를 따른다.

```txt
H1: 검색 의도가 분명한 제목
도입: 누구에게 필요한 글인지 설명
요약표: 추천 대상, 예산, 지역, 주의사항
360도 판단 기준: Tourpick360만의 차별화 기준
본문 H2 4개 이상
체크리스트
FAQ 3개 이상
최종 확인일
관련 글 3개 이상
```

Minimum per article:

```txt
본문 길이: 1,500자 이상
이미지: 1개 이상
내부 링크: 3개 이상
H2: 4개 이상
FAQ: 3개 이상
최종 확인일: 필수
```

### 3. First 30 Articles

#### Phase 1: 제주 집중 가이드 10개

Status: in progress, first 10 article structure is created.

Purpose:

- 사이트의 첫 전문 묶음
- 숙소 위치, 계절, 날씨, 동행 유형을 깊게 다룸
- Google bot이 사이트 주제를 이해하기 쉬움

#### Phase 2: 국내 계절 축제 여행 10개

Article candidates:

1. 봄꽃 축제 여행 숙소 위치 고르는 법
2. 여름 바다 축제 여행 준비 체크리스트
3. 가을 미식 축제 여행 동선 짜는 법
4. 겨울 빛 축제 여행에서 숙소 위치가 중요한 이유
5. 가족과 가기 좋은 국내 봄 축제 선택 기준
6. 커플 여행에 좋은 야간 축제 동선
7. 부모님과 가기 좋은 걷기 편한 축제 여행지
8. 비 오는 날에도 가기 좋은 실내형 축제와 대체 코스
9. 축제 기간 숙소 예약 전 확인해야 할 10가지
10. 지역 축제 여행 예산표 작성 방법

#### Phase 3: 국내 여행 준비 10개

Article candidates:

1. 국내여행 1박 2일 코스 짜는 법
2. 국내여행 2박 3일 동선 줄이는 방법
3. 렌터카 여행 숙소 위치 선택법
4. 대중교통 여행 숙소 위치 선택법
5. 가족여행 짐 체크리스트
6. 여행 예산표 작성 방법
7. 숙소 취소 규정 확인하는 법
8. 성수기 숙소 예약 순서
9. 비 오는 날 국내여행 대체 코스 짜는 법
10. 여행지 공식 정보를 확인해야 하는 이유

## Trust Signals

Must-have pages:

```txt
/about/
/contact/
/privacy/
/terms/
/disclosure/
/sitemap-index.xml
/rss.xml
/robots.txt
```

Footer must include:

- 소개
- 문의
- 개인정보처리방침
- 이용약관
- 광고/제휴 고지
- Sitemap
- RSS

## Search Engine Setup

After deployment:

```txt
1. Google Search Console domain property 등록
2. DNS TXT record로 소유권 확인
3. sitemap-index.xml 제출
4. rss.xml 제출 또는 robots.txt에서 발견되도록 유지
5. 주요 글 URL 색인 요청
6. Naver Search Advisor 등록
7. robots.txt 확인
8. sitemap 제출
9. 웹페이지 수집 요청
```

## Application Timing

AdSense 신청 조건:

```txt
[x] 정보성 글 20개 이상
[x] 가능하면 30개 이상
[ ] 주요 글 Google 색인 확인
[x] 빈 페이지 없음
[x] 404 링크 없음
[x] 모바일 레이아웃 정상
[x] About/Privacy/Contact 정상
[x] 카테고리별 글 수 부족해 보이지 않음
[ ] 제휴 링크 과다 없음
[ ] 광고 자리만 있는 박스 없음
```

During review:

```txt
2~3일에 글 1개 추가
기존 글 업데이트일 유지
검색 콘솔 색인 상태 확인
사이트 구조 변경 최소화
```

## What To Avoid Before Approval

```txt
해외여행 카테고리 추가
맛집만 따로 분리
상품 비교 페이지
제휴 링크 버튼 반복
광고 배치 테스트
내용 없는 축제 목록 페이지
자동 수집만 있고 에디터 설명이 없는 API 페이지
```
