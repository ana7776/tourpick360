# Tourpick360 AdSense Approval Plan

## Goal

Tourpick360의 1차 목표는 애드센스 승인이다. 승인 전까지는 제휴 링크, 수익형 도구, 과도한 광고 영역, 얇은 자동 생성 페이지를 넣지 않는다.

사이트 정체성은 다음 한 문장으로 고정한다.

> Tourpick360은 국내 여행지의 숙소, 동선, 준비 정보를 여행자가 판단하기 쉽게 정리하는 정보성 여행 가이드 사이트입니다.

승인 전 핵심 니치는 더 좁게 잡는다.

> 국내 계절 축제 여행 가이드

## Approval Priority

1. 원문 콘텐츠 30개 이상 확보
2. 각 글 1,200자 이상 작성
3. 각 글에 대표 이미지 1개 이상 사용
4. About, Contact, Privacy, Terms, Disclosure 페이지 유지
5. 명확한 카테고리 구조 유지
6. robots.txt, sitemap.xml 정상 제공
7. 빈 페이지와 얇은 카테고리 제거
8. 모바일 가독성 유지
9. 과도한 제휴/예약 유도 금지
10. 작성일, 최종 확인일, 참고 기준 표시

## Article Requirements

모든 글은 아래 구조를 기본으로 한다.

```txt
H1: 검색 의도가 명확한 제목
도입: 누구에게 필요한 글인지 설명
요약표: 지역, 대상, 예산, 주의사항
본문: 최소 H2 4개 이상
체크리스트: 예약/방문 전 확인사항
실전 팁: 일정, 교통, 계절 변수
FAQ: 3개 이상
최종 확인일
관련 글 3개 이상
```

## Image Policy

승인 전 이미지는 다음 중 하나를 사용한다.

1. 직접 촬영 이미지
2. 사용 허용 범위가 명확한 무료 이미지
3. 한국관광공사 등 공공데이터 이미지

모든 이미지는 alt 텍스트를 작성한다.

## What Not To Add Before Approval

승인 전에는 아래 기능을 넣지 않는다.

- 광고 위치 안내 박스
- 제휴 예약 버튼 과다 노출
- AI 문장 생성 도구
- 수익형 비교 계산기
- 얇은 태그 페이지
- 내용 없는 지역 페이지
- 외부 상품 링크만 모은 페이지
